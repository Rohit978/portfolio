A Quick Guide to Implementing a Custom Cursor

Published: September 24, 2025

Welcome to my first blog post! A custom cursor is a great way to add a unique, interactive element to a website. It provides a more personal touch and improves user experience by drawing attention to interactive elements. Here's a detailed look at how I implemented the one on my portfolio using just a little bit of HTML, CSS, and JavaScript.

1. The HTML Structure

The foundation of the effect requires two simple `div` elements placed at the top of your `body` tag. The first, `.cursor`, is the main, smaller cursor dot, and the second, `.cursor-follower`, is the larger, trailing element that creates the smooth hover effect.

<!-- Custom cursor -->
<div class="cursor"></div>
<div class="cursor-follower"></div>

2. The CSS Styling

The CSS is crucial for positioning and styling the cursors. We need to make them invisible to the user's mouse and ensure they stay on top of all other content.

.cursor, .cursor-follower {
    /* Ensures they stay in view regardless of scroll */
    position: fixed; 
    
    /* Makes the cursor always visible on top of other elements */
    z-index: 9999;
    
    /* Prevents the cursor from blocking clicks on elements beneath it */
    pointer-events: none;
    
    /* Initially centers them at the same point */
    top: 50%;
    left: 50%;
}

.cursor {
    width: 8px;
    height: 8px;
    background-color: #007bff;
    border-radius: 50%;
}

.cursor-follower {
    width: 40px;
    height: 40px;
    border: 1px solid rgba(0, 123, 255, 0.4);
    border-radius: 50%;
    transition: transform 0.15s ease-out; /* This creates the smooth, delayed motion */
}

3. The JavaScript Logic

The magic happens in JavaScript, where we track the mouse's position and update the cursors accordingly. The `requestAnimationFrame` function is used for performance, ensuring the animations are smooth and efficient.

const cursor = document.querySelector('.cursor');
const cursorFollower = document.querySelector('.cursor-follower');
let posX = 0, posY = 0;
let mouseX = 0, mouseY = 0;

function animateCursor() {
    // Calculates the distance between the mouse and the follower's current position
    posX += (mouseX - posX) / 9; 
    posY += (mouseY - posY) / 9;

    // Updates the follower's position smoothly
    cursorFollower.style.transform = `translate3d(${posX - 20}px, ${posY - 20}px, 0)`;
    cursor.style.transform = `translate3d(${mouseX - 4}px, ${mouseY - 4}px, 0)`;

    // Requests the browser to call the function again for the next frame
    requestAnimationFrame(animateCursor);
}

// Listens for mouse movement and updates the target positions
window.addEventListener('mousemove', e => {
    mouseX = e.clientX;
    mouseY = e.clientY;
});

// Starts the animation loop
animateCursor();
